# Shell Basics Project

This project contains basic shell scripts that I created while learning Unix shell commands.

Through these scripts, I practiced and learned how to:

- Navigate directories
- List files in different formats
- Work with hidden files
- Change between directories
- Copy files conditionally

## Scripts Included

- Display current working directory
- List directory contents
- List all files including hidden ones
- Navigate to previous directory
- List files in multiple directories
- Copy HTML files using conditions

## What I Learned

- Basic shell scripting structure (`#!/bin/bash`)
- Using commands like `ls`, `cd`, and `cp`
- File permissions (`chmod`)
- Working with hidden files
- Using relative and absolute paths
- Handling files and directories from the command line

This project helped me understand how the Unix shell works and how to automate simple tasks using scripts.
